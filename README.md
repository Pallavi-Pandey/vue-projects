# Vue Projects Collection

A collection of frontend applications built with Vue.js.

## Featured Projects

### [Recipe App](recipeapp/)
A comprehensive recipe search and management application.
- **Frontend**: Vue.js, Tailwind CSS, Vite.
- **Features**: API search, detailed meal views, and responsive design.

## How to Run
Navigate to a specific project directory (like `recipeapp`) to install dependencies and run the development server.
```bash
cd recipeapp
npm install
npm run dev
```
